# AI Legal Team — 9 AI Skills for Mid-Market Law Firms

**Run a smarter, leaner law firm with AI — without giving your data to a SaaS vendor.**

This library gives you **9 specialised AI skills** for mid-market law firms (10–80 solicitors). Each skill covers a core workflow — contract review, legal research, compliance, client communications, due diligence, litigation support, knowledge management, business development and practice operations — and works across **Kimi Code CLI**, **Claude Code** and **Codex CLI**.

No subscription. No vendor lock-in. Your data stays with you.

## What is the AI Legal Team?

The AI Legal Team is a set of reusable AI skills that turn your AI assistant into a legal specialist. Instead of writing long prompts for every task, you install a skill once and then say:

> "Run contract-analyst" or "/compliance-monitor"

The skill loads the workflow, rules, templates and jurisdiction-specific guidance automatically.

## How to install a skill in 30 seconds

### 1. Download the ZIP

Pick a skill from the table below and download the version for your assistant:

- **`-all.zip`** if you use more than one assistant
- **`-kimi.zip`** for Kimi Code CLI
- **`-claude.zip`** for Claude Code
- **`-codex.zip`** for Codex CLI

### 2. Unzip and copy one folder

**Kimi Code CLI:**

```bash
cp -R .kimi/skills/<skill-name> ~/.kimi/skills/
```

Then type in Kimi: `/<skill-name>`

**Claude Code:**

```bash
cp -R .claude/skills/<skill-name> ~/.claude/skills/
```

Then type in Claude: `<skill-name>`

**Codex CLI:**

```bash
cp -R .codex/skills/<skill-name> ~/.codex/skills/
```

Then type in Codex: `<skill-name>`

## Frequently asked questions

### What is an AI skill?
An AI skill is a reusable instruction pack that teaches your AI assistant how to perform a specific legal task with consistent quality.

### Do these skills work with Kimi, Claude and Codex?
Yes. Every skill has wrappers for Kimi Code CLI, Claude Code and Codex CLI.

### Is my data sent to third parties?
No. The skills run inside your AI assistant on your machine. Your data stays with you.

### Do I need to pay?
No. All skills are released under the MIT License.

### Can I use the same skill on multiple assistants?
Yes. The `-all.zip` file contains wrappers for Kimi, Claude and Codex.

### Do I need to code?
No. You download a ZIP, copy one folder, and invoke the skill by name.
