# Hartmut's AI Legal Team

**9 specialised AI agents for mid-market law firms.**

Open Source. Sovereign. Customisable.

## What is this?

A complete Agentic Team of 9 Claude Skills covering the key workflows of a mid-sized law firm (10–80 solicitors) — from contract analysis and compliance to business development.

No SaaS subscription. No vendor lock-in. Your data stays with you.

## The 9 Skills

| # | Skill | Function |
|---|-------|----------|
| 1 | **Contract Analyst** | Contract review, red flags, clause comparison (English Law + Swiss OR/ZGB) |
| 2 | **Legal Researcher** | Case law search, UKSC/BGer decisions, legislative tracking |
| 3 | **Compliance Monitor** | GDPR UK, SRA, nDSG, GwG, FINMA, EU AI Act — incl. risk heatmap |
| 4 | **Client Communicator** | Client letters, status updates, fee communications |
| 5 | **Due Diligence Assistant** | M&A, real estate, restructuring — checklists + findings |
| 6 | **Litigation Support** | Briefs, chronologies, evidence structuring, strategy memos |
| 7 | **Knowledge Manager** | Knowledge base, templates, precedent search, onboarding |
| 8 | **Legal BD & Marketing** | Firm newsletters, LinkedIn, events (SRA/BGFA compliant) |
| 9 | **Practice Operations** | Time recording, WIP, profitability, internal comms |

## Why Open Source?

Enterprise Legal AI (Harvey AI, CoCounsel) costs £800–2,000 per user/month. For 25 solicitors: **£250,000+ per year.**

This team costs a fraction — and you keep control of your data.

| | Enterprise SaaS | AI Legal Team |
|---|---|---|
| Setup | £8,000–40,000 | £15,000–25,000 |
| Annually (25 users) | £250,000+ | Token-based (low) |
| Data sovereignty | Cloud (often US) | Local / UK or Swiss hosting |
| Customisation | None | Full |

## Technology

- **Claude Skills** — Specialised AI agents on Anthropic Claude
- **MCP** (Model Context Protocol) — Open standard by the Linux Foundation
- **Apertus** — Swiss Open Source AI for data sovereignty
- **.skill format** — ZIP archives for Claude Desktop

## Installation

```bash
# Clone the repository
git clone https://github.com/hartmut-ux/ai-legal-team-en.git

# Load a single skill into Claude Desktop
# Drag & drop the .skill file or manually:
cp skills/contract-analyst/ ~/.claude/skills/
```
